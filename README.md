# Cors

- O Cors é uma single page application para gerenciamento de projetos, desenvolvida em reactJS.

# CLONE DO PROJETO

- Baixe o zip do projeto ou de um git clone na sua IDE

# INICIAR PROJETO

- npm install
- npm start
- npm backend
